1. У команды `id` есть опция, которая позволяет вывести на экран только id пользователя. Найдите эту опцию и выведите на экран id пользователя nobody.
2. Запишите получившуюся команду в файл solution.

- `id user_name`

## Solution

- `id -u nobody`
